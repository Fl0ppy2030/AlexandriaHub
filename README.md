# 📚 AlexandriaHub

Sistema de gerenciamento de biblioteca escolar desenvolvido com Flask + PostgreSQL.

## ✨ Funcionalidades

* 📖 Cadastro de livros
* 👨‍🎓 Cadastro de alunos
* 🔐 Sistema de login
* 📦 Empréstimo de livros
* ↩️ Devolução de livros
* 📝 Logs de movimentação
* 🔎 Busca dinâmica de livros
* 🖼️ Capas automáticas via API
* ☁️ Hospedado online com Render
* 🗄️ Banco de dados PostgreSQL (Supabase)

---

## 🛠️ Tecnologias Utilizadas

* Python
* Flask
* Flask SQLAlchemy
* Flask Login
* PostgreSQL
* Supabase
* Render
* HTML/CSS/JavaScript

---

## 🚀 Deploy Online

O sistema está hospedado utilizando:

* Frontend + Backend: Render
* Banco de dados: Supabase PostgreSQL
